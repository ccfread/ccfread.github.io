自学JAVA基础-CallBack回调-Hook-钩子，练习代码  

首先定义一个动物的接口，里面包含一个cry函数 
```java
package com.ezxmt.hook;

public interface Ianimal {
    void cry();
}
```

定义两个具体的动物狗和猫，实现动物接口

```java
package com.ezxmt.hook;

public class Cat implements Ianimal{

    @Override
    public void cry() {
        System.out.println("小猫猫叫声");
    }
}
```

```java
package com.ezxmt.hook;

public class Cat implements Ianimal{

    @Override
    public void cry() {
        System.out.println("小猫猫叫声");
    }
}
```java

    定义一个钩子，然后实现 
```java
 package com.ezxmt.hook;

public class Home {
    public static void buyAnimal(Ianimal a){
        //小动物叫唤
        a.cry();
    }

    public static void main(String[] args) {
        //买一只狗狗，狗狗就会叫一声
        buyAnimal(new Dog());
        //买一只猫猫，猫猫就会叫医生
        buyAnimal(new Cat());
    }
}
```


    这样就购买一只动物，这只动物的cry函数就会自动被实现，根据每只动物的特性，cry函数返回不一样  学习就暂时记录到这里
